Bicycle repair shop / online store

A modern serverless React / Firebase e-commerce and appointment booking application with Redux state management, PWA manifest, custom fonts, Google authentication, and Google Maps integration. Built with 2nd-generation Google Cloud Functions for Visma Pay payment processing and TriggerEmail for order tracking notifications. Includes product reviews and an easy to use dashboard for both users and administrators. Fully responsive.

Live Demo: https://bike26-2ffd8.web.app/
